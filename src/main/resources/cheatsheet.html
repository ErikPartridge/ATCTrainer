This file contains all of the commands for controling aircraft. All commands shall be issued over frequency via text.
Commands with aircraft selected:
CommandDescription
taxi PATH... [RUNWAY TO HS OF]Taxies an aircraft via the given route
stopStops an aircraft in less than two seconds
startResumes an aircraft's movement
cross RUNWAYCrosses specified runway
to [HEADING]Cleared takeoff, given heading, otherwise will proceed direct first fix
cancel [HEADING]Aircraft will cancel takeoff, or go around to assigned heading
luaw RUNWAYAircraft will line up and wait on the given runway
trs RUNWAYSets the runway transition for an aircraft on RNAV STAR specified in flight plan.
loc RUNWAY [HEADING]Turns the aircraft to given heading to intercept localizer for given runway, or flies present heading if not specified.
ils RUNWAY [HEADING]Same principle as localizer, but ILS
visual RUNWAY [HEADING]Turns towards [heading] or present and then tries visual. Must be within 15nm for visual approach.
haltWill hold present altitude, cancels VNAV guidance
dct NAVAID [ALTITUDE]Clears direct to a navaid, will cross at given altitude.
dctr NAVAIDDirect to a navaid, and then continues flight plan, must be on flight plan
alt ALTITUDEDescends or climbs to given altitude
tl HEADINGTurns left to specified heading
tr HEADINGTurns right to specified heading
rapp NAVAID RUNWAYProceeds direct NAVAID, then starts RNAV approach
sqk CODEWill squawk given code
sqcWill squawk mode C
sqidWill squawk ident
sqsWill squawk standby
